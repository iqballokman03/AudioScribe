# AudioScribe: AI-Powered Audio Intelligence

## Homepage Writing

### Hero Section
**Headline:** Turn audio into insight with incredible accuracy.
**Sub-headline:** Transcribe, translate, and summarize your meetings, podcasts, and recordings in seconds using the world's most advanced AI models.

### Key Value Propositions
1. **Intelligent Speaker Recognition**
   Never lose track of who said what. AudioScribe automatically identifies distinct speakers and formats them into a clean, readable dialogue.

2. **Multilingual by Design**
   Our engine detects the primary language of every segment. If it's not English, we provide an instant translation alongside the original text.

3. **Emotional Context**
   Go beyond the words. We detect the primary emotion—Happy, Sad, Angry, or Neutral—to give you the full context of every conversation.

4. **Temporal Precision**
   Every segment is timestamped. Click any timestamp to jump directly to that moment in the original audio recording.

5. **Instant Summarization**
   Don't have time to read the full transcript? Generate a concise, structured summary with a single click.

## Technical Specifications (The Tools)

### Core AI Engine
*   **Gemini 3.1 Pro**: We utilize Google's latest multimodal model to process raw audio files directly. This allows for superior accuracy in transcription, translation, and emotional semantic analysis.

### Frontend Architecture
*   **React 18 + Vite**: A high-performance, modern development environment for a snappy, single-page application experience.
*   **Tailwind CSS**: For a polished, responsive, and modern interface that works perfectly on mobile and desktop.
*   **Motion (Framer Motion)**: Fluid, purposeful animations that guide the user experience and handle state transitions smoothly.
*   **Lucide React**: A clean, consistent icon library for intuitive navigation.

### Capabilities & Exports
*   **Native Web Audio API**: Used for real-time browser recording and precise playback control.
*   **Multi-Format Export Engine**: Export your results to professional formats including:
    *   **TXT**: Plain text for quick notes.
    *   **DOCX**: Microsoft Word format for editing.
    *   **PDF**: Print-ready documents.
    *   **JSON**: Structured data for developers and integrations.
